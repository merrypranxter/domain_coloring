# Visual Targets

## 1. Rosette
Power function z^5, OKLCH wheel. A 5-petaled flower of light, phase wheel spinning smoothly, magnitude contours rippling outward. Clean, symmetrical, hypnotic.

## 2. Pole Bloom
Rational function (z^3-1)/(z^2+c), c=0.4+0.3i. Zeros bloom outward as bright spots, poles suck inward as dark vortices. Neon whirlpools. The most psychedelic of the set.

## 3. Zeta Strip
Riemann zeta function, view box Re∈[−5,40] Im∈[−20,20]. The critical strip glows. Famous and eerie. The phase wheel reveals the zeros as points where all hues converge.

## 4. Elliptic Tiling
Weierstrass elliptic function, half-periods (1, i). Doubly-periodic = auto-tiling wallpaper. Seamless, mathematical, beautiful.

## 5. Branch Neon
Log(z) with branch cut. The discontinuity at the negative real axis is lit hot magenta. The phase wrap is the art.
